# 1LINK Network Automation

Enterprise-style network automation project using:

- Ansible
- AWX
- iTop
- Cisco IOS
- Fortinet
- Palo Alto
- FMC

## Features

### Switch Automation

- VLAN Create/Delete
- Access Port Assignment
- Trunk Configuration
- Interface Management

### Router Automation

- Static Routes
- ACLs
- OSPF
- BGP

### Firewall Automation

- Fortinet Policies
- Palo Alto Policies
- FMC Rules

### Compliance

- NTP
- SSH
- Logging
- AAA
